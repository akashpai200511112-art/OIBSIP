# EDA on Retail Sales Data

## Description
This project is completed as part of the Oasis Infobyte Data Analytics Internship. It performs Exploratory Data Analysis (EDA) on a retail sales dataset to understand customer behaviour, product performance and sales trends.

## Objective
- Understand the dataset
- Perform exploratory data analysis
- Find sales trends
- Analyse customer demographics
- Generate business insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Retail Sales Dataset containing Transaction ID, Date, Customer ID, Gender, Age, Product Category, Quantity, Price per Unit and Total Amount.

## Analysis Performed
- Data inspection
- Missing value check
- Duplicate check
- Descriptive statistics
- Monthly sales analysis
- Product category analysis
- Gender analysis
- Age distribution
- Correlation heatmap

## Business Insights
- Monthly sales trends were analysed.
- Product category performance was identified.
- Customer demographics were studied.
- Correlation between numerical variables was visualised.

## How to Run
1. Open the notebook in Google Colab or Jupyter Notebook.
2. Upload the `retail_sales.csv` dataset.
3. Run all notebook cells.
