# Customer Segmentation Analysis

## Description
This project was completed as part of the Oasis Infobyte Data Analytics Internship. It uses the K-Means clustering algorithm to segment mall customers based on their annual income and spending score.

## Objective
- Perform customer segmentation
- Apply the K-Means clustering algorithm
- Identify different customer groups
- Generate business insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset
Mall Customer Segmentation Dataset

## Analysis Performed
- Data inspection
- Elbow Method
- K-Means clustering
- Cluster visualization
- Business insights

## How to Run
1. Open the notebook in Google Colab.
2. Upload `Mall_Customers.csv`.
3. Run all cells.
